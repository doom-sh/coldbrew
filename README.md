## Introduction

A visual interface to quickly install your favorite macOS apps from Homebrew Cask

## Stack

- Vercel
- Next.js
- Tailwind CSS
- Prisma
- Vercel/Neon Postgres
- TRPC
- TypeScript

## Local development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
